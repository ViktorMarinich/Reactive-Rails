#### *Install*
1.Open terminal window in *root* folder and run:

```sh
a)bundle install
b)rake db:setup
c)rails s
```
2.Open next terminal window in *frontend* folder and run:

```sh
a) npm install `or` sudo npm install
b) npm start
```
3.Wait to start all servers

4.Visit http://localhost:8080
